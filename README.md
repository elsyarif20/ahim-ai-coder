# 💻 AHIM CODE - AI Assistant

**AHIM CODE** adalah aplikasi asisten koding berbasis AI yang dibangun menggunakan **Streamlit** dan ditenagai oleh **Groq API** untuk performa inferensi super cepat.

## ✨ Fitur Utama
- **Ultra Fast Response**: Menggunakan infrastruktur LPU dari Groq.
- **Modern UI**: Antarmuka dark mode yang bersih dan profesional.
- **Model Selection**: Pilihan model terbaru seperti Llama 3.3, Mixtral, dan Gemma.
- **Secure**: Integrasi API Key yang aman.
